# Employee Salary Prediction Using Machine Learning

## Overview
This project uses Machine Learning to predict whether an employee belongs to a higher salary category based on experience and age. A Decision Tree Classifier is trained and evaluated using real-world machine learning workflow steps.

## Features
* Data preprocessing using Pandas
* Train-Test Split
* Decision Tree Classification
* Model Evaluation
* Accuracy Score Calculation
* Confusion Matrix Visualization

## Technologies Used
* Python
* Pandas
* Matplotlib
* Scikit-Learn

## Dataset Attributes
* Experience
* Age
* Salary_High (Target Variable)

Target Values:
* 0 = Lower Salary Category
* 1 = Higher Salary Category

## Machine Learning Process
1. Load Dataset
2. Select Features and Target
3. Split Data into Training and Testing Sets
4. Train Decision Tree Model
5. Make Predictions
6. Evaluate Accuracy
7. Visualize Results with Confusion Matrix

## How to Run
Install dependencies:
pip install -r requirements.txt

Run the project:
python salary_prediction.py
